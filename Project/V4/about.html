About Project: V1

Author: Ashton Krause
Course: CIS155 Web Developement 1
Date: April 25th, 2021

This website is a project meant to showcase my ability to code a functional website using html and css. My topic for this website is a space for my sister, Briar Rose Krause, to showcase her digital art, and to take commissions for new art.
All of the requirements that have been implemented in version 1 of my website are as follows (hover to expand):

Module Level Objectives
M1LO6: Demonstrate the use of basic HTML tags to create a simple Web page (HTML, this item should be specified once, describing how the file it is in meets this outcome)
M1LO7: Demonstrate the use of ordered lists (HTML)
M1LO8: Demonstrate the use of external style sheets (HTML, note the external style sheet must exist and contain CSS)
M1LO9: Demonstrate the use of inline images (HTML)
M1LO10: Demonstrate the use of special characters from extended character sets (HTML, use entities, not cut-and-pasted unicode characters)
M1LO12: Create navigation using unordered lists (HTML)
M1LO13: Demonstrate how CSS can change presentation of textual elements (CSS, these are styles that impact text that is rendered on the page)
M2LO14: Demonstrate and explain the use of progressive enhancement in CSS (CSS, see 2-6 Employing Progressive Enhancement)
M2LO15: Apply colors using CSS (CSS, see 2-5 Working with Color in CSS)

The web pages the Homepage, About Project: V1, Gallery, and About are where the html requirements are implemented. The respective html files are index.html, about.html, gallery/gallery.html, and aboutbriar.html. the web pages have not been fleshed out yet, as well as Contact and Commissions are mostly empty. The CSS requirements are in the file styles.css.
Right now this website is limited in its capabilities. Right now my website implements virtually everything we have covered in the class. It was enough to get my website presentable and partially functional as an early build.
